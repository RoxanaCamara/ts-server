## What need install to start ts-server?

```bash
npm i -g typescript
npm i -g nodemon
```

## How start ts-server?

First command to compile and other console to execute

```bash
tsc --watch
nodemon dist/app.js
```

## What version of node use?

```bash
v18.15.11
```
